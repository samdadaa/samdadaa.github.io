Zahl-Raten

GitHub

Dieses Projekt ist ein einfaches "Zahl raten"-Spiel, das mit Java und der Swing-Bibliothek für die grafische Benutzeroberfläche (GUI) entwickelt wurde. Das Spiel präsentiert dem Benutzer eine zufällige Zahl zwischen einem bestimmten Bereich und fordert ihn auf, diese zu erraten.
